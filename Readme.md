# Plotando Gráficos
- Usamos o Pandas, NumPy e o MatPlotLib;
- IO através de arquivos CSV.
- Tratamento de dados:
    - Correção de tipo;
    - Criação de multi-índice;
    - Enriquecimento com cópia de valores ordenados.
